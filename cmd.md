# Подсказки по командной строке

Команда смены дериктории
```sh
cd \folder_name
```

Команда отображения текущей дериктории для Mac0s, Linux
```sh
pwd
```

Листинг текущей дериктории Windows
```sh
dir
```

Linux, Mac0s
```sh
ls
```

Удаления файла в Windows
```sh
del <filename>
```
в Linux, Mac0s
```sh
rm <filename>
```
